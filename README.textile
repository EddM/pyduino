h1. pyduino

A quick a dirty python web app to allow for the control of an LED through a browser.

Very limited functionality so far, but I plan to add more to it in the future.

h2. Prerequisites

* Python (>= 2.6)
* "pyserial":http://pyserial.sourceforge.net/
* "juno":http://github.com/breily/juno

h2. To Make it Do Stuff

Easy:

<pre>
<code>
  python pyduino_srv.py
</pre>
</code>
